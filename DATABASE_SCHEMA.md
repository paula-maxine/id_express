# ID Express - Database Schema & API Reference

## 📊 Firestore Collections Structure

### 1. `users` Collection
Stores user profile information

**Document ID**: `{uid}` (Firebase Auth UID)

**Fields**:
```
{
  "email": string           // User's email address
  "fullName": string        // User's full name
  "createdAt": timestamp    // Account creation date
}
```

**Example**:
```json
{
  "email": "john@example.com",
  "fullName": "John Doe",
  "createdAt": 1707033600000  // Feb 4, 2024
}
```

**Security Rules**:
- Read: User can only read their own document
- Write: User can only write their own document
- Delete: Not allowed in demo

**Indexes**: Auto-created (uid as primary key)

---

### 2. `documents` Collection
Stores document registration records

**Document ID**: Auto-generated by Firestore

**Fields**:
```
{
  "uid": string             // Document owner's UID (ref to users)
  "documentType": string    // Type: "National ID" | "Passport" | "Driver License"
  "idNumber": string        // ID number from document
  "fullName": string        // Name from document
  "imageUrl": string        // URL to image in Firebase Storage
  "createdAt": timestamp    // Upload timestamp
  "status": string          // "pending" | "approved" | "rejected"
}
```

**Example**:
```json
{
  "uid": "user_abc123",
  "documentType": "National ID",
  "idNumber": "123456789",
  "fullName": "John Doe",
  "imageUrl": "https://storage.googleapis.com/id-express.appspot.com/users/user_abc123/documents/National%20ID/1707037200000.jpg",
  "createdAt": 1707037200000,
  "status": "pending"
}
```

**Security Rules**:
- Read: User can only read their own documents
- Write: User can only write their own documents
- Create: Authenticated users can create documents
- Delete: Not allowed in demo

**Indexes**:
- Primary: uid (for user-specific queries)
- Optional: createdAt (for sorting)

## 📁 Firebase Storage Structure

### Path Structure
```
users/
└── {uid}/
    └── documents/
        ├── National ID/
        │   ├── 1707033600000.jpg
        │   └── 1707037200000.jpg
        ├── Passport/
        │   └── 1707040800000.jpg
        └── Driver License/
            └── 1707044400000.jpg
```

### File Naming
- Format: `{timestamp}.jpg`
- Example: `1707033600000.jpg`
- Prevents conflicts and preserves upload order

### URL Format
```
https://storage.googleapis.com/{bucket}/users/{uid}/documents/{docType}/{filename}.jpg
```

## 🔗 Service Methods Reference

### AuthService

#### `signUp(email, password, fullName)`
Register a new user

**Parameters**:
- `email`: string (email format)
- `password`: string (min 6 chars)
- `fullName`: string

**Returns**: `Future<UserCredential?>`

**Example**:
```dart
await authService.signUp(
  email: 'john@example.com',
  password: 'SecurePass123',
  fullName: 'John Doe',
);
```

**Errors**: 
- "weak-password"
- "email-already-in-use"
- "invalid-email"

---

#### `login(email, password)`
Authenticate existing user

**Parameters**:
- `email`: string
- `password`: string

**Returns**: `Future<UserCredential?>`

**Example**:
```dart
await authService.login(
  email: 'john@example.com',
  password: 'SecurePass123',
);
```

**Errors**:
- "user-not-found"
- "wrong-password"
- "invalid-email"

---

#### `logout()`
Sign out current user

**Returns**: `Future<void>`

**Example**:
```dart
await authService.logout();
```

---

#### `currentUser` (Getter)
Get current logged-in user

**Returns**: `User?` (null if no user)

**Example**:
```dart
User? user = authService.currentUser;
String uid = user?.uid ?? '';
```

---

### FirestoreService

#### `saveUserProfile(uid, email, fullName)`
Create user profile in Firestore

**Parameters**:
- `uid`: string (from Auth)
- `email`: string
- `fullName`: string

**Returns**: `Future<void>`

**Example**:
```dart
await firestoreService.saveUserProfile(
  uid: 'user123',
  email: 'john@example.com',
  fullName: 'John Doe',
);
```

**Creates**: Document in `users` collection

---

#### `saveDocumentRecord(uid, documentType, idNumber, fullName, imageUrl)`
Save document upload record

**Parameters**:
- `uid`: string
- `documentType`: "National ID" | "Passport" | "Driver License"
- `idNumber`: string
- `fullName`: string
- `imageUrl`: string (URL from Storage)

**Returns**: `Future<void>`

**Example**:
```dart
await firestoreService.saveDocumentRecord(
  uid: 'user123',
  documentType: 'National ID',
  idNumber: '123456789',
  fullName: 'John Doe',
  imageUrl: 'https://storage.googleapis.com/...',
);
```

**Creates**: Document in `documents` collection

---

#### `getUserDocuments(uid)`
Fetch all documents for a user

**Parameters**:
- `uid`: string

**Returns**: `Future<List<Map<String, dynamic>>>`

**Example**:
```dart
List<Map<String, dynamic>> docs = 
  await firestoreService.getUserDocuments('user123');

// Result:
// [
//   {
//     'documentType': 'National ID',
//     'idNumber': '123456789',
//     'status': 'pending',
//     ...
//   }
// ]
```

**Query**: `documents.where('uid', '==', uid).orderBy('createdAt', desc)`

---

#### `getUserProfile(uid)`
Fetch user profile

**Parameters**:
- `uid`: string

**Returns**: `Future<Map<String, dynamic>?>`

**Example**:
```dart
Map<String, dynamic>? profile = 
  await firestoreService.getUserProfile('user123');

// Result: { 'email': '...', 'fullName': '...', ... }
```

---

### StorageService

#### `uploadDocumentImage(uid, documentType, imageFile)`
Upload image to Firebase Storage

**Parameters**:
- `uid`: string
- `documentType`: "National ID" | "Passport" | "Driver License"
- `imageFile`: File (from ImagePicker)

**Returns**: `Future<String>` (download URL)

**Example**:
```dart
String url = await storageService.uploadDocumentImage(
  uid: 'user123',
  documentType: 'National ID',
  imageFile: selectedFile,
);
// Returns: 'https://storage.googleapis.com/...'
```

**Path**: `users/{uid}/documents/{documentType}/{timestamp}.jpg`

**Features**:
- Automatic compression
- Organized folder structure
- Returns public download URL

---

#### `deleteDocumentImage(imageUrl)`
Delete image from Storage

**Parameters**:
- `imageUrl`: string (full URL)

**Returns**: `Future<void>`

**Example**:
```dart
await storageService.deleteDocumentImage(
  'https://storage.googleapis.com/...',
);
```

## 🔍 Query Examples

### Get User's National IDs
```dart
List<Map<String, dynamic>> nationalIds = 
  (await firestoreService.getUserDocuments(uid))
    .where((doc) => doc['documentType'] == 'National ID')
    .toList();
```

### Get Pending Documents
```dart
List<Map<String, dynamic>> pending = 
  (await firestoreService.getUserDocuments(uid))
    .where((doc) => doc['status'] == 'pending')
    .toList();
```

### Count User Documents
```dart
int count = (await firestoreService.getUserDocuments(uid)).length;
```

## ⚡ Performance Notes

- Firestore reads/writes: < 100ms typical
- Image upload: 1-5 seconds (depends on size)
- Query by uid: O(n) where n = user documents
- Download images: <1 second typical

## 🔐 Compliance

- User IDs are isolated (no cross-user access)
- Images stored in user-specific folders
- Firestore rules enforce user ownership
- No sensitive data in error messages
